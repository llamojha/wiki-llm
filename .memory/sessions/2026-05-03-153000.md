## Key Decisions

- **Storage = AWS S3 only** — no MinIO/R2 abstraction, standard S3 SDK
- **Editing = in-browser only** for MVP 1
- **`index.md` mandatory** — machine-maintained, ask-wiki agent's catalog
- **Generated separation = single `generated/` prefix**, `source_type` metadata for provenance
- **Auth = single user** for OSS; Keycloak/OIDC ships only with SaaS
- **Ask-wiki agent provider = Amazon Bedrock Nova 2 Lite** (`amazon.nova-2-lite-v1:0`, 1M context)
- **Ask-wiki agent scope = tool-using agent** (search/read/propose-page) with citations + refusal
- **Out of scope:** vector search, autonomous writes, multi-agent
- **Routing = state-driven SPA** (not `/[...slug]` routes) — matches prototype, deferred to Phase 2 for deep-linking
- **`'use client'` = boundary-only** — only AppShell + components with hooks at module scope
- **Doc type = discriminated union** `AuthoredDoc | GeneratedDoc` — needed for TS strict narrowing on `doc.generated`
- **Doc-bodies port format = mirror prototype source** (single-line JSX, no `{' '}` spacers) — for byte-comparable parity
- **Theme bootstrap inline script** — no-flash dark/light on reload
- **Date hydration = useState + useEffect** — avoid SSR/CSR drift on `new Date()`
- **Parity = automated source diff + Playwright pixel diff** (not eyeball sign-off)
- **TweaksPanel deferred** — dev-only, low value, revisit if needed
