# Session: Upload & Ingest UI for Phase 3

**Date:** 2026-05-14 00:18–00:53 CEST

## Summary

Built the portal UI for Phase 3 (Ingest Pipeline) — an UploadPanel component that lets users upload .md files to a space, process pending raw files via Bedrock, and trigger re-indexing. Added streaming progress (NDJSON) so the UI shows a real-time progress bar during multi-file curation. Ran a code review and fixed the medium-severity issues.

## Key Decisions

- Upload + Process + Re-index all live in a single `UploadPanel` modal (opened via 📎 button in sidebar)
- Curate endpoint supports `{ stream: true }` for NDJSON progress streaming
- New `/api/raw?space=X` endpoint returns pending file count (shown on "Process all" button)
- New `/api/reindex` endpoint for standalone index regeneration without Bedrock processing
- No auto-ingest toggle in UI (env var only per spec)
- Single file upload only (spec constraint)
