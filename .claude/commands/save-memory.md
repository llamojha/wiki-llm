# save-memory

Save a summary of this conversation session to memory.

## Instructions

1. Summarize the current conversation into a concise markdown document with:
   - **Date**: Current date/time
   - **Summary**: 2-3 sentence overview of what was discussed
   - **Key Decisions**: Bullet list of decisions made
   - **Outcomes**: What was accomplished (files created, problems solved, etc.)
   - **Open Items**: Any unfinished tasks or follow-ups

2. Save to `.memory/sessions/` with filename format: `YYYY-MM-DD-HHMMSS.md`

3. Confirm the save with the filename.

Keep the summary brief but useful for future context recall.
